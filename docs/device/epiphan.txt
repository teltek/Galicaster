Plugin Configuration
--------------------

Admitted values:
    name: Name assigned to the device.
    device: Device type: vga2usb
    flavor: Matterhorn "flavor" associated to the track. (presenter|presentation|other)
    location: Device's mount point in the system (e.g. /dev/video0).
    file: The file name where the track will be recorded.
    drivertype: Wheter the device use a v4l2 or a v4l interface, to guarantee backwards compatibility (v4l2|v4l)
    As for Ubuntu 10.10 or similar use v4l.
    resolution: Output resolution. If the input resolutino doesn't match the output, the input will be scaled respecting the ratio - by introducing black borders.
    framerate: Output framarate. If the input framerate doesn't match the output, frames will be dropped or duplicated accordingly.

Examples
--------

[track1]
name = DVI2PCIe
device = epiphan
flavor = presentation
location = /dev/screen
file = SCREEN.avi
resolution = 1024,768
framerate = 10/1
drivertype = v4l2
